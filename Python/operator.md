# 操作符

**运算操作符**：
10/3 = 3.3333 
% 取余
\*\*：幂 2**3 = 8
//： 只有整数 地板除法：10 // 3 = 3 
没有++和-- 只能用+=或者-=
可以连等： a = b = c = d = 10

**比较操作符**： > >= < <=  3<4<5：合理
**逻辑操作符**： and or not
not > and > or

1.and是在布尔上下文中从左到右计算表达式的值；
2.0、''、[]、()、{}、None、False在布尔上下文中为假；其它任何东西都为真；
3.如果布尔上下文中的某个值为假，则返回第一个假值；
4.所有值都为真，则返回最后一个真值。

三元操作符：
If x < y:
    small = x
Else:
    small = y
可以简化为：
Small =  x if x < y else y

## 运算顺序：
幂 > 正负号 > 算数操作 > 比较 > 逻辑
